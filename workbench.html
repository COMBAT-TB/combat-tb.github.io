combatTB and combat-SARS-Cov-2 Workbench

The combatTB project has produced two Workbench platforms, for analysis of M. tuberculosis (Mtb) and SARS-CoV-2 sequence data. The first of these, for analysis Mtb data, is described in this publication in mSphere. Code for deploying the combatTB Workbench can be found here.

The work on the combatTB Workbench led to the development of a similar Workbench for SARS-CoV-2. The code for deploying the "combat-SARS-Cov-2 Workbench" can be found here. Funding from the Africa CDC Institute for Pathogen Genomics (IPG) allowed us to deploy an instance of the Workbench at workbench.sanbi.ac.za for use by labs supported by the IPG's Pathogen Genomics Initiative.
